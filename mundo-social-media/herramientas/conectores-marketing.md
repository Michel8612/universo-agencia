# Conectores de Marketing Disponibles

Estos conectores MCP están disponibles en esta sesión de Claude Code.
Cuando los actives con tu cuenta, Claude podrá usarlos directamente.

## Conectores Detectados

| Conector | Para qué sirve | Activar con |
|---------|---------------|-------------|
| **Canva** | Crear diseños, thumbnails, posts con IA | `mcp__plugin_marketing_canva__authenticate` |
| **Notion** | Base de datos de contenido, calendario editorial | `mcp__plugin_marketing_notion__authenticate` |
| **Slack** | Notificaciones internas, alertas de la agencia | `mcp__plugin_marketing_slack__authenticate` |
| **Figma** | Diseños de UI para proyectos de clientes | `mcp__plugin_marketing_figma__authenticate` |
| **Klaviyo** | Email marketing avanzado (alternativa a Mailchimp) | `mcp__plugin_marketing_klaviyo__authenticate` |
| **Ahrefs** | SEO: keywords, backlinks, análisis competencia | `mcp__plugin_marketing_ahrefs__authenticate` |
| **Amplitude** | Analytics de producto, comportamiento usuarios | `mcp__plugin_marketing_amplitude__authenticate` |
| **Supermetrics** | Agregar datos de todas las plataformas de ads | `mcp__plugin_marketing_supermetrics__authenticate` |

## Cuáles activar primero (por impacto en la agencia)

1. **Canva** → genera imágenes para posts sin necesitar diseñador
2. **Notion** → calendario editorial y CRM básico gratis
3. **Ahrefs** → imprescindible para ofrecer SEO a clientes
4. **Slack** → para que la agencia te notifique internamente

## Cómo activar
Dile a Claude Code: "activa el conector de Canva" y lo hace automáticamente.
