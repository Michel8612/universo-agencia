# MUNDO SEO — Posicionamiento Orgánico

Genera tráfico gratuito para tus clientes. El SEO bien hecho es el mejor ROI del marketing.

## Modelo
- Estrategia SEO, análisis de competencia: **Sonnet**
- Meta descriptions, alt texts, contenido en masa: **Haiku**

## Servicios SEO

### SEO Técnico (setup único 800-2,000€)
- Auditoría técnica completa (velocidad, Core Web Vitals, indexación)
- Corrección de errores críticos
- Estructura de URLs optimizada
- Schema markup (rich snippets)
- Sitemap + robots.txt

### SEO de Contenidos (retainer 500-1,500€/mes)
- 4-8 artículos/mes optimizados para keywords objetivo
- Investigación de keywords de baja competencia y alta intención
- Optimización de contenido existente
- Link building básico

### SEO Local (para negocios físicos 300-800€/mes)
- Google Business Profile optimizado
- Citas locales (directorios)
- Reseñas automatizadas
- Keywords geolocalizadas

## Herramientas (gratuitas o low-cost)
- **Google Search Console** — datos reales de Google (gratis)
- **Ahrefs Webmaster Tools** — versión gratis limitada
- **Screaming Frog** — auditoría técnica (gratis hasta 500 URLs)
- **Ubersuggest** — keywords y competencia (freemium)
- **PageSpeed Insights** — velocidad (gratis)

## Proceso Auditoría SEO
1. Análisis de posicionamiento actual (qué keywords rankea ya)
2. Análisis de competidores top 3 en el nicho
3. Identificar keywords con oportunidad (volumen alto, competencia baja)
4. Plan de contenidos para los próximos 3 meses
5. Fix de errores técnicos críticos
6. Producción de contenido con `mundo-contenido/`

## KPIs que reportar al cliente
- Posición media en Google Search Console
- Clics orgánicos mensuales (evolución)
- Keywords en top 10 / top 3
- Tráfico orgánico vs mes anterior
