# MUNDO E-COMMERCE — Tiendas Online

Construye y optimiza tiendas que venden. Mercado enorme: todo negocio quiere vender online.

## Modelo
- Arquitectura de tienda, integraciones complejas: **Sonnet** / **Opus** si es grande
- Copy de productos, descripciones en masa: **Haiku**

## Servicios

### Tienda Nueva (1,500-8,000€)
Según complejidad:
- **Básica** (hasta 50 productos, Shopify): 1,500-2,500€
- **Media** (hasta 500 productos, Next.js + Stripe): 3,000-5,000€
- **Avanzada** (marketplace, B2B, ERP integrado): 5,000-15,000€

### Optimización de Tienda Existente (500-2,000€)
- Auditoría de conversión (CRO)
- Mejora de velocidad (Core Web Vitals)
- Optimización de fichas de producto
- Setup de recuperación de carritos

### Gestión Mensual (300-1,000€/mes)
- Nuevos productos / actualizaciones
- Campañas de email + ads coordinadas
- Reporte de ventas mensual

## Stack Tecnológico

### Opción 1 — Shopify (recomendada para empezar)
- Fácil de gestionar para el cliente
- Apps para todo
- Precio: 29-299$/mes (el cliente paga)
- Nuestro trabajo: configuración + tema personalizado

### Opción 2 — Next.js + Stripe (más control, más trabajo)
- 100% personalizable
- Sin comisiones de plataforma
- Integra con cualquier sistema
- Stack: Next.js + Stripe + Supabase + Vercel

### Opción 3 — WooCommerce (para clientes con WordPress)
- Gratis (plugin)
- Ecosistema enorme
- Requiere más mantenimiento

## Integraciones Clave para E-commerce
- **Pagos**: Stripe, PayPal, Bizum (España), Klarna (pago fraccionado)
- **Envíos**: MRW, Correos, DHL, Packlink
- **Facturación**: Holded, Factusol, Sage
- **Analytics**: GA4 + Meta Pixel + TikTok Pixel

## IA Aplicada al E-commerce (diferenciador)
- Descripciones de producto generadas por IA (con Haiku = muy barato)
- Recomendaciones personalizadas basadas en historial
- Chatbot de atención con `mundo-chatbots/`
- Predicción de demanda para gestión de stock
- Copy de anuncios personalizado por segmento
